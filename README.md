# AI Mock Interview Platform

An AI-powered full-stack interview preparation platform that helps users practice technical and HR interviews with intelligent AI-generated questions and answer evaluation.

---

## 🚀 Live Demo

### Frontend

[https://interview-ai-platform.vercel.app](https://interview-ai-platform.vercel.app)

### Backend

[https://interview-ai-platform-1.onrender.com](https://interview-ai-platform-1.onrender.com)

---

# ✨ Features

* 🔐 JWT Authentication
* 🤖 AI-generated interview questions
* 🧠 AI answer evaluation and feedback
* 📚 Interview history tracking
* 🎨 Modern responsive UI
* ⚡ Fast Vite + React frontend
* ☁️ Full deployment using Vercel & Render

# 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=5000

MONGO_URI=your_mongodb_connection_string

JWT_SECRET=your_secret_key

GROQ_API_KEY=your_groq_api_key
```

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/interview-ai-platform.git
```

---

## 2️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

## 3️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

# ▶️ Run Project

## Start Backend

```bash
cd backend
npm run dev
```

---

## Start Frontend

```bash
cd frontend
npm run dev
```
